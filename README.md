# Automated Data Cleaning Script

### Project Overview
This Python script automates the **ETL (Extract, Transform, Load)** process for raw production datasets. It replaces manual Excel data entry, ensuring data integrity for analysis.

### Key Features
* **Data Validation:** Detects and filters corrupt entries.
* **KPI Calculation:** Computes average metrics (e.g., Temperature).
* **Error Handling:** Separates valid records from anomalies.

### Technologies
* Python 3.x
* CSV Library
